# memo-draw

## Getting started

Install the dependencies :

```bash
npm install
```

Start the development sever :

```bash
npm run dev
```

## Run tests

```bash
npm test
```

## Run eslint check
```bash
npx eslint . --ext .js,.jsx,.ts,.tsx
```
